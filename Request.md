# Agentflow VS Code 外掛需求文件

## 專案概述

開發一個 VS Code 外掛，名稱為 **Agentflow**，用於自動格式化 `.js` 檔案的程式碼內容。

---

## 功能需求

### 1. 基礎 JavaScript 格式化

- 以標準 JavaScript 格式化規則為主要依據
- 支援縮排、空格、括號等一般 JS 格式規範
- 可於儲存檔案時自動觸發格式化，或透過指令手動觸發

### 2. 自訂語法支援：`{???:` 開頭的物件格式

- 允許物件使用**任意字串**（包含英文、中文或其他字元）作為鍵名（不加引號），並在鍵名後直接換行，格式如下：

  ```js
  {ALL:
    ...
  }

  {初始化:
    ...
  }

  {任意鍵名:
    ...
  }
  ```

- 格式化過程中，**不得**將此類鍵名視為錯誤或強制加上引號
- 應保留鍵名 `:` 之後的換行與內容縮排結構
- 適用規則：只要符合 `{<任意字串>:` 後接換行的模式，均視為合法格式予以保留

### 3. 多個空行自動壓縮為單一空行

- 當程式碼中出現連續兩行以上的空白行時，自動將其壓縮為一個空白行
- 適用範圍：整個 `.js` 檔案內容
- 範例：

  **格式化前：**
  ```js
  const a = 1;



  const b = 2;
  ```

  **格式化後：**
  ```js
  const a = 1;

  const b = 2;
  ```

---

## 適用範圍

- 僅針對副檔名為 `.js` 的檔案進行格式化處理

---

## 非功能需求

- 外掛應輕量，不影響 VS Code 整體效能
- 格式化結果需具可預測性與一致性
